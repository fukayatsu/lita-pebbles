# lita-pebbles

A Lita handler for fun :)

## Installation

Add this line to your lita's Gemfile:

    gem 'lita-pebbles'

And then execute:

    $ bundle

## Usage

```
Lita: sudden 突然の死
Lita: flip foobar
Lita: omikuji
Lita: misawa
...
```
